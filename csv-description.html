Constructors:

CsvBuilder(DataSet dataset)

  • Description: Private constructor that initializes a CsvBuilder object with a given DataSet.
  • Parameters:
    • dataset: The DataSet containing the data to be processed.

Properties:

ValueRenderEvent (Event)

  • Description: Gets or sets the value rendering event. Must be set before calling the Build method.
  • Type: event ValueParser?
  • Access: Public
  • Exceptions: Throws an InvalidOperationException if modified after the Build method is called.

Methods:

public static ICsvBuilder Datasets(params DataTable[] dataTables)

  • Description: Static factory method to create an instance of CsvBuilder from an array of DataTables.
  • Parameters:
    • dataTables: Array of DataTable objects.
  • Returns: An instance of ICsvBuilder.

public static ICsvBuilder Datasets(params List<string>[] rows)

  • Description: Static factory method to create an instance of CsvBuilder from an array of string lists.
  • Parameters:
    • rows: Array of string lists.
  • Returns: An instance of ICsvBuilder.

public ICsvExtractor Build(params int[] columnsTobePresentedForTableIndex)

  • Description: Constructs the CSV data based on the provided DataSet and optional array of column indices.
  • Parameters:
    • columnsTobePresentedForTableIndex: Optional array of column indices to be included.
  • Returns: An instance of ICsvExtractor.
  • Exceptions: Throws an InvalidOperationException if ValueRenderEvent is modified after the Build method is called.

public void Dispose()

  • Description: Releases resources. Disposes of the MemoryStream.

public MemoryStream GetStream()

  • Description: Gets the MemoryStream containing the CSV data after flushing the StreamWriter.
  • Returns: The MemoryStream.

public void SaveAsFile(string filePath)

  • Description: Saves the CSV data to a file specified by the given file path.
  • Parameters:
    • filePath: The path where the CSV data will be saved.
